# 遇见成都 —— 静态网页项目介绍

## 一、项目概述

本项目是一个以"成都旅游"为主题的静态展示网页，采用纯 HTML + CSS + 原生 JavaScript 构建，无需任何第三方框架或依赖。页面通过简洁清晰的布局和轻量交互，向访客展示成都的特色文化、热门景点和出行建议。

## 三、页面结构与功能说明

### 3.1 导航栏（Navbar）

- **位置**：页面顶部，始终吸顶可见
- **功能**：提供"首页 / 特色体验 / 热门景点 / 出行建议"四个锚点链接，点击后平滑滚动到对应区域

**实现方式**：
- 使用 `position: sticky` 固定导航栏在页面顶部
- `backdrop-filter: blur()` 实现毛玻璃半透明背景效果
- 页面平滑滚动通过 CSS 一行代码实现：`html { scroll-behavior: smooth; }`，无需 JavaScript

### 3.2 首屏横幅（Hero Banner）

- **内容**：渐变背景 + 副标题 + 主标题 + 描述 + CTA 按钮
- **视觉**：多层渐变叠加半透明遮罩，营造成都温暖悠闲的氛围感

**核心交互 —— 打字机效果**（JavaScript 实现，约 15 行代码）：
- 主标题"来成都，感受慢生活与烟火气"在页面加载后逐字显示
- 使用 `setTimeout` 递归调用，每 80ms 显示一个字符
- 先将标题文字清空，再逐字符追加回 DOM，形成打字动画

```javascript
const h1 = document.querySelector('.typewriter');
const text = h1.textContent;
h1.textContent = '';
let i = 0;
const type = () => {
  if (i < text.length) {
    h1.textContent += text[i++];
    setTimeout(type, 80);
  }
};
setTimeout(type, 400);
```

### 3.3 特色体验推荐

- 三栏网格布局（CSS Grid），展示三项成都特色体验：
  - 川剧变脸
  - 鹤鸣盖碗茶
  - 深夜火锅
- 每张卡片包含 emoji 图标、标题、描述和建议标签

### 3.4 热门景点推荐

- 三列两行共 6 张景点卡片，使用 CSS Grid 布局
- 每张卡片包含：背景图片、景点名称、简介、星级评分
- **hover 动效**：鼠标悬停时卡片上浮并加深阴影，同时背景图片放大 1.05 倍，纯 CSS 实现

```css
.spot-card:hover {
  transform: translateY(-6px);
  box-shadow: 0 14px 32px rgba(0,0,0,0.13);
}
.spot-card:hover .card-image { transform: scale(1.05); }
```

**图片处理**：
- 所有图片通过 CSS `background-image` 设置，`background-size: cover` 保证比例一致
- 同时设置了 fallback 背景色，图片加载失败或离线时仍有视觉占位

### 3.5 出行建议

- 三栏卡片布局
- 内容涵盖：最佳季节、交通方式、必吃推荐
- 卡片 hover 时上浮 + 阴影加深

### 3.6 响应式设计

通过 CSS Media Queries 在移动端自动适配：

| 断点 | 布局调整 |
|------|----------|
| ≤768px | 导航栏垂直排列；三列网格变为单列；导航链接换行居中 |
